# Исследование лимитов провайдеров LLM

**Дата:** 2026-06-24
**Метод:** анализ документации + live API headers + тестовые запросы

---

## Вывод: источника правды о лимитах через API нет

Ни один из трёх провайдеров не отдаёт информацию о лимитах/балансе через API:
- `/v1/usage`, `/v1/billing`, `/v1/subscription` — все возвращают 404 или 403
- `/v1/models` не содержит информации о лимитах
- Единственный способ — анализировать **заголовки ответа** (rate limit headers)

---

## Mistral

### Источник данных
Официальная документация: https://docs.mistral.ai/admin/user-management-finops/tier

### Лимиты Free Tier
- Rate limit: 1 запрос в секунду
- Назначение: evaluation, prototyping
- Trial credits: $25-$50 для новых аккаунтов
- Стартап-программа: до $30K credits

### Лимиты Production (из документации)
- Зависят от плана (Scale, Enterprise)
- Контакт: sales@mistral.ai

### Проверка через API
- Заголовки ответа: **отсутствуют** (Mistral не возвращает rate limit headers)
- 429 Too Many Requests: при превышении 1 req/sec
- Единственный способ проверки — документация + тестирование

### Наши ключи
- 2 ключа, оба Free Tier
- Работают с интервалом ≥1 сек между запросами

## Cohere

### Источник данных
Официальная документация: https://docs.cohere.com/docs/rate-limits

### Лимиты Trial (бесплатный)
- **100 вызовов всего** (не в месяц — всего!)
- Rate limit: 20 req/min на модель
- Ежемесячный лимит: 1000 вызовов (но trial = 100 всего)

### Лимиты Production
- 500 req/min на модель (Command A, R, R7B)
- Для Command A+ и Reasoning: 20 req/min, требуется контакт sales@cohere.com

### Проверка через API (заголовки ответа)
Cohere возвращает лимиты в заголовках:
```
x-endpoint-monthly-call-limit: 1000
x-trial-endpoint-call-limit: 100
x-trial-endpoint-call-remaining: 98
```

### Наши ключи
- 4 ключа, все **Trial** (не Production!)
- Осталось: **98 из 100** вызовов (после моих тестов)
- ⚠️ Критично: использовать экономно, не тратить на тесты

### Формат ответа Cohere v2/chat
```json
{
  "message": {
    "content": [
      {"type": "thinking", "thinking": "..."},
      {"type": "text", "text": "ответ"}
    ]
  }
}
```

## Cerebras

### Источник данных
Официальная документация: https://inference-docs.cerebras.ai/quickstart
Документация по лимитам: https://inference-docs.cerebras.ai/support/rate-limits (недоступна — 404)
Прайсинг: https://cerebras.ai/pricing

### Лимиты Free Tier (из сторонних источников)
- **1M tokens/день** бесплатно (с апреля 2026)
- Модели: Llama 4 Scout, Qwen3 32B
- Контекст: 8,192 tokens (до 128K по запросу)
- Скорость: 2,600+ tokens/sec
- Без кредитной карты

### Лимиты Developer ($50/мес)
- Generous rate limits
- Доступ к gpt-oss-120b и zai-glm-4.7

### Проверка через API
- Заголовки ответа: **отсутствуют**
- 429 Too Many Requests: при превышении лимита
- Баланс/usage endpoint: не существует

### Наши ключи
- 5 ключей, все рабочие
- Ключи 3 и 4 иногда получают rate limit при частом тестировании
- Тип аккаунта: неизвестен (предположительно Developer или Free)

## Рекомендации

1. **Cohere**: ключей осталось 98/100. Использовать только для критичных задач, не тратить на тесты
2. **Mistral**: единственный полностью бесплатный провайдер без лимита вызовов (только rate limit 1 req/sec)
3. **Cerebras**: для проверки баланса нужно зарегистрироваться в Cloud Console (cloud.cerebras.ai)
4. **Общее правило**: для точной информации о лимитах нужно доступ к Dashboard провайдера, а не только API
